# 狼人杀信息记录主控台 WerewolfManager

本项目是一个用于**线下狼人杀游戏**的主控台管理软件，基于 Python + PyQt6 开发。适合法官（主持人）高效管理每晚流程、白天发言、玩家身份分配及游戏日志等。

## 功能特色

- **角色管理**：支持多种狼人杀身份和技能展示，便于法官查阅。
- **玩家信息表**：自动随机分配角色，显示每位玩家的编号、身份、状态和描述（死亡原因/备注），支持状态/描述实时编辑。
- **夜间行动记录**：一键记录每晚所有行动，便于溯源和复盘。
- **白天发言关键词收集**：法官可收集白天每位玩家发言关键词，自动保存到日志。
- **发言顺序管理**：支持顺序、逆序和随机起点发言（自动排除死亡玩家），实用高效。
- **日志总览与清空**：所有操作自动汇总，便于法官事后查证或清空重置。
- **多套身份板子**：支持常见狼人杀12人局自定义身份搭配。
